The active sequence is Rabimodulated.xml with the microwave frequency swept from 3.825 GHz to 3.925 GHz. The sequence enables channels [1 1 1 0 0 0], sets the microwave/IQ source, performs an initial polarization and detection, then applies a modulated Rabi pulse and performs a second detection. Because full_expt is 0, the optional "1 level reference" block is skipped; therefore readout 1 is the initial polarized/reference readout and readout 2 is the post-microwave-pulse readout. The sequence file sets length_rabi_pulse to 52 ns and mod_depth to 1.

The first/readout reference is comparatively smooth with only modest scan-to-scan variation. The second/post-pulse readout shows a strong, localized dip centered around roughly 3.875-3.880 GHz, dropping from the high-30-count baseline to about 27 counts in the combined data. The same dip appears in both per-average traces for readout 2, while readout 1 does not show a matched common-mode dip of comparable size. That frequency-localized, readout-role-specific contrast is consistent with a pODMR resonance.

The provided sequence is Rabimodulated.xml with mw_freq swept from 3.825 GHz to 3.925 GHz. The active instructions first acquire the true 0-level reference by polarizing and detecting, then, because full_expt = 0, skip the 1-level reference block. The active experimental readout is therefore the detection after rabi_pulse_mod_wait_time using length_rabi_pulse = 5.2e-08 s (52 ns) and mod_depth = 1, followed by detection. Thus readout 1 is the 0-level/reference readout and readout 2 is the microwave-pulse signal readout.

The reference readout remains relatively flat over the scan, while the signal readout shows a pronounced and localized decrease around 3.875-3.880 GHz, reaching about 39-40 counts versus a typical baseline near 47-49 counts. The dip is visible in the combined data and in the per-average overlay, so it is not just a single average artifact. This behavior is consistent with a pODMR resonance being present.

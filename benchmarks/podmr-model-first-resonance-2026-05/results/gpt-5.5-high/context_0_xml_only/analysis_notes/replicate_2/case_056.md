The active sequence is Rabimodulated.xml. It sets a microwave frequency sweep and uses a 52 ns rabi_pulse_mod_wait_time pulse with mod_depth = 1, followed by detection. The sequence first polarizes and detects a true 0-level reference. Because full_expt = 0, the optional 1-level reference block is skipped. The later detection after the modulated microwave pulse is therefore the pODMR measurement readout.

The two combined raw readouts both show a broad downward drift across the frequency sweep, so a resonance decision should not rely on raw count level alone. The pulsed readout has its sharpest low point around 3.875 GHz, and the contrast against the reference readout is also locally negative there. That dip is visible in both per-average pulsed readouts, although the reference channel also has some simultaneous fluctuation. Overall this supports a weak but present pODMR resonance rather than only smooth baseline drift.

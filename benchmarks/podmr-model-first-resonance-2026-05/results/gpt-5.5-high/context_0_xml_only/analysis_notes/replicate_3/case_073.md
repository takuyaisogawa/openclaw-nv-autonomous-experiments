The provided sequence XML defines a Rabimodulated pODMR-style scan with mw_freq varied across the exported points. The active variables in the XML give length_rabi_pulse = 5.2e-08 s, so the microwave pulse duration is 52 ns after sample-rate rounding, and mod_depth = 1. The instructions first polarize and detect a true 0-level reference, then wait. The optional 1-level reference block is disabled because full_expt = 0, so the second active detection is the signal after rabi_pulse_mod_wait_time using the 52 ns pulse and mod_depth 1.

Using those readout roles, readout 1 is the 0-level reference and readout 2 is the pulsed pODMR signal. The raw traces are noisy with only two averages, but the pulsed signal divided by the reference has a localized dip near 3.87-3.88 GHz. The combined readout2/readout1 ratio reaches about 0.957 at 3.870 GHz and 0.936 at 3.880 GHz, and both individual averages show a depression at those same frequencies, especially around 3.880 GHz. Other points fluctuate, so the feature is not clean, but the frequency-localized and partially repeatable contrast is more consistent with a present pODMR resonance than with a flat absent response.

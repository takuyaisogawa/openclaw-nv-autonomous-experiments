The provided sequence XML is Rabimodulated.xml. With full_expt = 0, the active acquisition is a true m_S = 0 reference readout after polarization, followed by a single Rabi-modulated microwave pulse and then the signal readout. The inactive m_S = +1 reference branch is skipped. Thus readout 1 is the m_S = 0 reference and readout 2 is the post-pulse pODMR signal.

The relevant pulse is length_rabi_pulse = 52 ns with mod_depth = 1. Using the supplied setup calibration, the Rabi frequency is about 10 MHz at this modulation depth, so the pulse is close to a pi pulse on resonance. If the microwave frequency hit the addressed transition, the second readout should show a clear fluorescence drop relative to the first, with an expected scale that could approach the stated 22 percent m_S = 0 to m_S = +1 contrast.

The measured combined signal/reference differences are not consistent with that. The largest negative combined excursions are only about -5 to -6 percent, while several positive excursions are comparable or larger, including around 3.840 GHz and 3.910-3.915 GHz. The per-average overlay shows substantial tracking/baseline changes between the two stored averages, so the apparent excursions do not form a reliable repeatable resonance feature. Since the near-pi pulse should make a resonance appear as a strong readout-2 dip and no such stable dip is present, I classify this case as resonance absent.

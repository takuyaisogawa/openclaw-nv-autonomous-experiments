Active sequence: Rabimodulated.xml, varying mw_freq from 3.825 GHz to 3.925 GHz in 5 MHz steps. The XML applies adj_polarize followed by detection for the first readout, so readout 1 is the true 0-level/reference readout with no microwave pulse immediately before detection. Because full_expt = 0, the optional 1-level reference block is skipped. The active signal block then applies rabi_pulse_mod_wait_time with length_rabi_pulse = 52 ns and mod_depth = 1, followed by detection, so readout 2 is the microwave-pulse readout.

The raw readouts are noisy and only two averages are present, but the microwave-pulse readout shows a consistent suppression relative to the reference over a contiguous high-frequency region near 3.905-3.920 GHz. The contrast values there are negative by several percent and the effect is not just a single isolated point; it rebounds at the last sweep point. This pattern is compatible with a pODMR resonance in the swept band.

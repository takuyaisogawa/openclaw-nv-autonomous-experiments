The active sequence is Rabimodulated.xml with mw_freq swept from 3.825 GHz to 3.925 GHz. The active pulse is rabi_pulse_mod_wait_time before the second detection, using length_rabi_pulse = 52 ns and mod_depth = 1; full_expt = 0 disables the optional 1-level reference block. Thus readout 1 is the initial true 0-level reference after polarization and detection, while readout 2 is the measurement readout after the modulated Rabi pulse.

The combined raw readouts show a distinct readout-2 depression centered near 3.875-3.880 GHz, reaching about 29-30 counts while neighboring points recover to about 34 counts. This dip is not just a monotonic drift feature: it is localized in the swept microwave frequency and is strongest in the post-pulse readout role. The per-average overlay shows substantial average-to-average drift, but the combined post-pulse trace still contains a clear frequency-localized contrast feature consistent with a pODMR resonance.

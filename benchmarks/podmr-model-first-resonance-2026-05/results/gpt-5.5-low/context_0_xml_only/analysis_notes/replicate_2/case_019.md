The active sequence is Rabimodulated.xml with mw_freq swept from 3.825 GHz to 3.925 GHz. The sequence first performs optical polarization and detection, giving the first readout as the bright/reference level. Because full_expt is 0, the optional 1-level reference block is not executed. The active measurement then applies rabi_pulse_mod_wait_time with length_rabi_pulse = 52 ns and mod_depth = 1 before the final detection, so the second readout is the microwave-affected signal readout.

The first readout remains mostly near 40-42 counts across the sweep and does not show a strong central feature. The second readout has a pronounced, reproducible drop to about 32 counts around 3.875-3.880 GHz in both averages, then recovers on either side. Since this dip appears in the microwave-affected readout and not in the reference readout, it is consistent with a pODMR resonance.

Sequence XML identifies the active sequence as Rabimodulated with mw_freq scanned from 3.825 to 3.925 GHz. The variables used for this case have length_rabi_pulse = 52 ns and mod_depth = 1. The full_expt variable is 0, so the optional 1-level reference block is inactive. The actual readout order is therefore: first detection after optical polarization as the 0-level/reference readout, then a modulated Rabi pulse followed by the signal detection. Thus readout 1 should be treated as the reference and readout 2 as the post-pulse pODMR signal.

The raw traces are noisy with only two averages, but the post-pulse signal relative to the reference shows a repeated depression in the 3.87-3.885 GHz region. The combined r2/r1 ratio reaches its deepest value at about 3.880 GHz, and both individual averages show reduced normalized signal near 3.880 GHz, even though other smaller fluctuations are present elsewhere. Because the feature is aligned with the swept microwave frequency and appears in the role-corrected signal/reference comparison, I judge a pODMR resonance to be present.

The active sequence is Rabimodulated.xml with mw_freq swept from 3.825 GHz to 3.925 GHz. The sequence first polarizes and detects a true m_S = 0 reference, then skips the separate m_S = +1 reference because full_expt is 0, then applies rabi_pulse_mod_wait_time with length_rabi_pulse = 52 ns and mod_depth = 1 before the second detection. Thus the two stored readouts should be interpreted as the 0-state reference and the post-microwave signal readout, not as a full 0/1 reference pair.

Using the stated setup facts, mod_depth = 1 gives about a 10 MHz Rabi frequency, so a 52 ns pulse is close to a pi pulse. If the swept microwave frequency crossed a real pODMR resonance, the post-pulse readout should show a substantial fluorescence reduction relative to the 0 reference, approaching the roughly 22 percent contrast scale for this setup. Instead, the combined signal-reference difference averages only about -0.53 raw units on a roughly 47-count signal, with point-to-point excursions of a few counts and no stable, broad, or isolated resonance-shaped depression. The deepest individual differences are much smaller than the expected near-pi-pulse contrast and are comparable to apparent tracking/noise variations across the two stored averages.

Decision: resonance_absent.

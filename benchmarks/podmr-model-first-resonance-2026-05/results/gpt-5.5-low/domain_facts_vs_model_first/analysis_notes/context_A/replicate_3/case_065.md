The active sequence is Rabimodulated.xml with mw_freq swept from 3.825 to 3.925 GHz in 5 MHz steps. The XML instructions first acquire a polarized m_S = 0 optical reference with no microwave pulse, then, because full_expt = 0, skip the separate m_S = +1 reference branch and acquire the actual signal after a rabi_pulse_mod_wait_time pulse followed by detection. Thus readout 1 is the polarized reference and readout 2 is the post-microwave-pulse readout.

The exported active variables give length_rabi_pulse = 52 ns and mod_depth = 1. With the provided setup scale of about 10 MHz Rabi frequency at mod_depth = 1, this is approximately a pi pulse, so a real resonance should produce a noticeable reduction in the post-pulse fluorescence relative to the reference. The raw traces are noisy and have only two stored averages, which should not be overinterpreted as independent repeatability, but the combined readouts show the strongest normalized drop at 3.895 GHz: readout 2 / readout 1 is about 0.908 there, a roughly 9.2% reduction. Neighboring points partially recover, and this feature is larger than most point-to-point fluctuations. The size is below the full 22% m_S = 0 to m_S = +1 contrast scale, but plausible for a pODMR response under these conditions.

Decision: resonance_present.

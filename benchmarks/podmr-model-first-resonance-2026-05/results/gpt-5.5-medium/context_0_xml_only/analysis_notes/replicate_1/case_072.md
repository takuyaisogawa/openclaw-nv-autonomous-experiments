The active sequence is Rabimodulated.xml with mw_freq as the scan variable from 3.825 to 3.925 GHz in 5 MHz steps. The XML sets length_rabi_pulse to 5.2e-08 s, and after sample-rate rounding at 250 MHz this remains a 52 ns pulse. The exported variable values set mod_depth to 1. The instructions first polarize and detect before the microwave pulse, so readout 1 is the true 0-level/reference readout. Because full_expt is 0, the optional 1-level reference block is skipped. The active measurement pulse is then rabi_pulse_mod_wait_time with the 52 ns pulse and mod_depth 1, followed by the second detection, so readout 2 is the post-pulse signal readout.

The raw readouts have drift and only two averages, so the important check is the signal relative to the reference rather than either raw trace alone. The readout2-readout1 contrast shows the strongest coherent negative feature at 3.885-3.890 GHz, about -3.23 counts or roughly -6.9 percent at both adjacent points. This dip is present relative to the reference readout and is visible in the combined raw traces. There are other noisy excursions, including later negative points, but the adjacent negative contrast around 3.885-3.890 GHz is consistent with a pODMR resonance response for the active pulse sequence.

The provided sequence is Rabimodulated.xml with mw_freq swept from 3.825 to 3.925 GHz. The active instructions set the microwave and then perform a polarization/detection readout for the true 0-level reference, wait, skip the optional 1-level reference because full_expt is 0, apply rabi_pulse_mod_wait_time, and detect again. Therefore readout 1 is the bright 0-level reference and readout 2 is the post-microwave signal readout.

The active microwave pulse is the modulated Rabi pulse with length_rabi_pulse = 5.2e-08 s. At the 250 MHz sample rate this is exactly 13 samples, so the rounded active pulse duration remains 52 ns. The variable values give mod_depth = 1, so the active pulse is fully modulated.

To decide on resonance I compared the post-pulse signal readout against the reference, not just the raw readout labels. The strongest and most plausible ODMR feature is near 3.895 GHz: combined readout 2 is 45.385 while the reference readout 1 is 50.000, giving a signal/reference ratio of about 0.908. Both individual averages also show a strong negative contrast at this point, with ratios about 0.900 and 0.915. Neighboring points are noisier and the full trace is not a smooth ideal line shape, but the repeated post-pulse depression at 3.895 GHz relative to the reference is large enough to indicate a pODMR resonance.

Active sequence: Rabimodulated.xml. The XML sets full_expt = 0, so the optional "1 level reference" block is inactive even though do_adiabatic_inversion is true. The active measurement performs polarization, then a detection that acts as the m_S = 0 reference, waits, applies one modulated Rabi pulse, and performs the second detection as the post-pulse signal.

The provided XML has mod_depth = 1 and length_rabi_pulse = 5.2e-08 s. With the stated setup calibration, mod_depth = 1 gives about a 10 MHz Rabi frequency, so a 52 ns pulse is approximately a pi pulse. If the swept microwave frequency hit a pODMR resonance, the second readout should show a clear fluorescence reduction relative to the m_S = 0 reference, on the order of the 22% contrast scale.

The combined readouts do not show such a dip. The post-pulse readout is usually close to or above the reference, with only small isolated negative differences. Looking within the stored averages, the large upper/lower offset is consistent with tracking cadence rather than independent repeatability; within each average the post-pulse/reference ratio remains near unity and never shows a consistent pi-pulse-scale drop. There is therefore no convincing pODMR resonance in this scan.

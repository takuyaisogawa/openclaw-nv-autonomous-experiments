Active sequence and readout roles:

- The provided `inputs/sequence.xml` is `Rabimodulated.xml`, scanning `mw_freq`.
- The active instruction path first runs `adj_polarize`, then `detection`, so readout 1 is the true mS=0 fluorescence reference.
- `full_expt = 0`, so the optional 1-level reference block is skipped; `do_adiabatic_inversion` is not used on the active path.
- The active pODMR signal is then generated by `rabi_pulse_mod_wait_time(PSeq, sample_rate, length_rabi_pulse, mod_depth, switch_delay, ch_on)` followed by `detection`, so readout 2 is the post-microwave-pulse signal readout.
- `mod_depth = 1`. `length_rabi_pulse = 5.2e-08 s`; at `sample_rate = 250 MHz`, rounding gives 13 samples, still 52 ns.

Quantitative model:

For a square resonant Rabi pulse, using the supplied setup facts, the Rabi frequency is about 10 MHz at `mod_depth = 1`. With detuning `d` in Hz and Rabi frequency `fR` in Hz, the driven mS=+1 population is

`P1(d) = (fR^2 / (fR^2 + d^2)) * sin^2(pi * t * sqrt(fR^2 + d^2))`.

The expected fluorescence ratio of the post-pulse readout to the mS=0 reference is

`R2/R1 = 1 - C * P1(d)`, with contrast `C = 0.22`.

Using `fR = 10 MHz`, `t = 52 ns`, and `C = 0.22`:

- On resonance, `P1(0) = sin^2(pi * 10e6 * 52e-9) = 0.9961`.
- Expected on-resonance ratio is `1 - 0.22 * 0.9961 = 0.7809`.
- Expected on-resonance drop is `21.91%` of the mS=0 readout.

Observed normalized data:

- The largest normalized drop is at scan value 3.880 GHz.
- There, readout 1 is 21.3462 and readout 2 is 16.9808.
- The observed ratio is `16.9808 / 21.3462 = 0.7955`, i.e. a `20.45%` drop.
- For the same readout 1 level, the model expectation on resonance is `21.3462 * 0.7809 = 16.67`, close to the observed 16.98.

I also fit only the resonance center using the finite-pulse model above with fixed contrast and fixed Rabi frequency. The best center was 3.8772 GHz. The sum of squared residuals for this model on the normalized ratios was 0.0183, compared with 0.0824 for a flat no-resonance ratio model, a 4.5x improvement.

Decision:

The post-pulse readout has a frequency-localized dip of the correct magnitude for the 52 ns, `mod_depth = 1` pulse. Tracking drift is visible in the stored averages, so I do not treat the two averages as a strong repeatability test, but the reference-normalized combined readouts match the quantitative Rabi-pulse resonance model. I decide that a pODMR resonance is present.

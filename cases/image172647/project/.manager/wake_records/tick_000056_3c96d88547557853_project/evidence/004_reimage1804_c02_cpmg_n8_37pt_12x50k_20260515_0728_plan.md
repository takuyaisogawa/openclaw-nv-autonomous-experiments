# reimage1804_c02 CPMG N=8 weak-13C discriminator plan (20260515_0728)

## Why this is the next non-blind test

Terminal det=1.5, det=1.0, and det=1.25 MHz Ramsey data support a short/few-us T2star order and leave a weak det-shift-consistent high-sideband candidate, but not a claim-grade Ramsey-only 13C result. The target amplitudes are about-percent/sub-percent and low-sideband/direct-Larmor/carrier support is incomplete. A different-protocol discriminator is justified before closing the 13C branch.

## CPMG.xml timing/readout review

The validated route is `auto__cpmg` / `CPMG.xml`. The active XML does:

- polarize and detect a true-0 reference;
- apply a pi pulse and detect an ms=1 reference;
- apply pi/2, then repeat `wait tau`, pi, `wait tau` for `N` loops, then a final pi/2;
- detect the final CPMG echo signal.

So adjacent pi pulses are separated by about `2*tau`, and total free evolution is about `2*N*tau`. Readout 3 should be treated as the candidate signal after terminal metadata verification; readout 2 is not the default signal for this sequence.

## Explicit target model

Using the terminal weak-pi pODMR center 3876501336.754 Hz gives B = 0.035914 T and expected 13C Larmor f13 = 384.6 kHz. For `CPMG.xml` timing:

- toggling-function convention `f ~= 1/(4*tau)` gives tau = 0.650 us;
- interpulse convention `tau = 1/(2*f)` gives tau = 1.300 us.

The chosen scan is tau 0.45..1.60 us, 37 points, step 31.9 ns. It samples the two convention targets near points 7 and 28. With N=8, the rough target widths tau/N are 81.3 ns and 162.5 ns, so this is a scout-quality but still targeted grid.

## Shot/drift choice

Initial 59-point preview gave 839.8 s per average, too close to daytime drift limits. The selected 37-point, 12 x 50000 plan keeps the MATLAB advisory preview at 536.2 s per average and 6459 s total, while preserving 600k shots/point. The shot floor is about 0.129%, so a 1% feature is 7.7x the binomial floor before drift/systematics. Recent det=1.25 Ramsey had no scan-order drift flags and healthy pre/post counts (44.073/42.669 kcps), but the plan still requires fresh verifier, queue-idle check, and MATLAB advisory before execute.

## Planned execution if gates pass

- Sequence: `auto__cpmg` / `CPMG.xml`.
- Settings: `N=8`, tau 0.45..1.60 us, 37 points, 12 averages x 50000 reps, `mw_freq=3876501336.754 Hz`, `length_pi=52 ns`, `mod_depth=1`, `mw_ampl=-5`, `ampIQ=5`, `freqIQ=50 MHz`, `do_adiabatic_inversion=false`.
- Expected review: raw export/plot, CPMG readout-role verification, scan-order-aware drift diagnostic, target-region checks around 0.650 us and 1.300 us, plus off-target baseline/reference checks.

This is not a blind repeat. It should strengthen, downgrade, or keep explicitly ambiguous the weak Ramsey 13C candidate.

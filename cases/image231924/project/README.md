# Project Snapshot

Sanitized snapshot of the cold-archived OpenClaw project folder for the
`image231924` case. The goal is to preserve the project state, evidence
records, notes, reports, and bridge-facing records as closely as possible while
excluding credentials and disabling live hardware execution.

This directory includes the sanitized root project files, `.manager/` records,
`work/` notes and artifacts, `work/bridge_jobs/` records, `summaries/`,
`experiment_intents/`, and the archived `data_git_20260512/` package
containing case-linked MATLAB and bridge artifacts. A small number of transient
lock/cache files were intentionally skipped.

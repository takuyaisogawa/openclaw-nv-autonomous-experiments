# Safety Boundary

The public repository is designed for audit and case-data inspection.

Included:

- project manager core;
- scheduler wake records;
- backlog, state, evidence, and intent handling;
- submit-spec validation;
- source copies of the case-referenced Python project-management, batch-run,
  bridge-watch, and direct-enqueue layers, with public execution disabled;
- a public summary of the project-manager prompt context.

Not included:

- direct instrument control as a runnable public entry point;
- live queue mutation as a runnable public entry point;
- lab-specific device configuration;
- MATLAB live-lab backend source;
- private operator channels;
- automatic startup configuration.

The production source copies are present for audit. Direct execution entry
points are disabled, and private live queue roots/configuration are excluded.
